# 知识库内容 Agent 产品化与定价方案

这套系统可以做成独立产品，也可以做成可安装 Skill / Agent Pack，还可以做高客单价定制。关键不是卖一个排版网页，而是卖“把客户私有知识变成持续获客内容”的工作流。

当前 MVP 只保留两条产品线：文章获客版和视频分发版。素材接入和评分共用，输出结果分开，暂时不做大而全内容平台。

## 两条产品线

### 1. 文章获客版

目标用户：咨询顾问、培训老师、B2B 服务商、企业知识管理服务商、垂类 IP、销售型创始人。

核心能力：

- 输入主题后自动调用 Obsidian、飞书云文档、上传文档、图片、外部热点和行业监控。
- 自动选择该进入文章的知识点、案例、反常识判断、行动清单和业务转化点。
- 输出公众号文章、标题组、摘要、封面文案、发布前预测日志。
- 自动评分：传播分、获客分、信息增量、证据强度、可执行性、转化自然度。
- 排版后复制到公众号后台，或创建公众号草稿，用户手机端确认发布。

### 2. 视频与多平台分发版

目标用户：需要用短视频、长视频和社交平台获客的人。

核心能力：

- 同一个主题生成抖音口播脚本、YouTube 长视频结构、X/Twitter thread、朋友圈短文。
- 自动生成 3 秒开场、冲突点、案例证据、转场、屏幕演示提示和 CTA。
- 把文章或脚本压缩成图片卡片：标题、副标题、3-5 个要点、二维码/CTA。
- 图片可复制、下载到电脑或手机，用于朋友圈、社群、小红书或公众号配图。

## 产品形态

- Web/SaaS：最容易给普通客户用，适合收费和权限管理。
- 桌面安装包：适合重视本地隐私的知识工作者，可用 Tauri/Electron 封装。
- Skill 包：适合装进 Codex、Claude Code 或其他 Agent 编程工具，让用户在自己的工作区调用。
- CLI/MCP 服务：作为跨平台适配层，给 WorkBuddy、Qoder、KimiCode 等工具接入；不同工具只写薄适配器。
- 定制交付：给客户接入自己的飞书、Obsidian、监控源、发布流程和复盘报表。

## 定价建议

3 万、5 万可以收，而且方向是对的，但必须按“定制内容生产系统”报价，不要按“网页工具”报价。

- 自助版：一次性 199-499 元，或 99-299 元/月。
- Pro 版：999-2999 元/月，或 5000-15000 元/年，包含多平台脚本、监控和评分。
- 轻定制：3 万起，适合一个知识源、一个主输出渠道、一个审批流程。
- 标准定制：5 万-8 万，适合飞书 + Obsidian + 行业监控 + 公众号草稿 + 视频脚本 + 图片卡片 + 培训。
- 深度定制：10 万以上，适合团队多人协作、CRM/企微/数据看板、私有化部署、合规审计。
- 运维校准：3000-10000 元/月，负责监控源维护、Prompt/Rubric 迭代、平台规则变化、月度复盘。

## 报价边界

报价前必须确认：

- 客户有几个知识源：飞书、Obsidian、本地文件、网盘、Notion、企微聊天等。
- 每个知识源是否有官方 API，是否需要用户授权，是否允许本地同步。
- 要监控哪些外部来源：AI 新闻、竞品、公众号、YouTube、X/Twitter、行业网站。
- 输出哪些渠道：公众号、抖音、视频号、YouTube、X/Twitter、朋友圈、社群。
- 系统只创建草稿，还是允许用户确认后自动发布。
- 是否需要图片卡片、长图、封面图、视频分镜和字幕。
- 是否要记录发布后的数据并做 T+7/T+30 复盘。

## 交付版本

### MVP 交付

- 素材上传/粘贴/链接登记。
- 主题驱动生成文章。
- 内容评分和预测日志。
- 公众号排版和富文本复制。
- 图片卡片导出。

### 标准交付

- 飞书授权读取文档/知识库。
- Obsidian 本地库索引和召回。
- 外部热点监控 Agent。
- 文章、视频脚本、社交短文多格式输出。
- 公众号草稿创建，用户手机端确认。

### 高级交付

- 客户私有知识库向量索引。
- 多人审批流。
- 自动复盘发布数据。
- 专属评分 Rubric。
- 私有化部署或桌面端安装包。

## 技术路线

- 前端：素材接入台、写作工作台、评分面板、排版预览、图片卡片导出。
- 后端：Source Agent、Knowledge Agent、Trend Agent、Writing Agent、Scoring Agent、Publish Agent。
- 连接器：Feishu/Lark API、Obsidian 本地文件索引、网页/RSS/YouTube/X 监控、微信公众号草稿接口。
- 分发：Web SaaS、Tauri/Electron 桌面包、Codex/Claude Code Skill、CLI/MCP 跨工具适配。
