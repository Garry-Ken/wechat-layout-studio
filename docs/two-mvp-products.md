# 两个最小 MVP 产品

当前只做两个最小产品，不做大而全平台。两者共用素材接入，但输出、评分和交付边界分开。

## 1. 文章获客 MVP

一句话：输入一个主题，从私有知识和外部素材里重组出一篇可发布、可获客的公众号文章。

必须有：

- 主题、目标读者、业务主线。
- Obsidian / 飞书 / 上传文档 / 素材链接入口。
- 自动生成公众号文章。
- 传播分、获客分、综合分和预测日志。
- 公众号排版预览、富文本复制、HTML 导出。
- 图片卡导出，方便朋友圈和社群转发。

暂不做：

- 多账号矩阵管理。
- 全自动发布。
- CRM 线索管理。
- 大型团队协作。

## 2. 视频分发 MVP

一句话：输入同一个主题，从知识库素材生成一套视频和社交分发包。

必须有：

- 抖音 / 视频号 60 秒口播。
- YouTube 8 分钟结构。
- X / Twitter Thread。
- 朋友圈文案。
- 图片卡文案和 PNG 图片卡导出。
- 发布前预测日志。

暂不做：

- 自动剪辑。
- 自动配音。
- 自动字幕。
- 多平台一键发布。
- 复杂数据看板。

## 共用底座

- Source Agent：接入文档、图片、链接和私有知识源。
- Knowledge Agent：召回客户问题、案例、方法论和交付经验。
- Scoring Agent：做发布前预测，发布后可 T+7 复盘校准。

## 收费入口

- 自助版先卖文章获客 MVP。
- Pro 版增加视频分发 MVP。
- 定制版只承诺接入客户自己的知识源、监控源和审批流程，不承诺做成通用内容平台。
